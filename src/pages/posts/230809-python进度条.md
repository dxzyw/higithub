这是一种新型的进度条，具有实时吞吐量，预计到达时间和非常酷的动画！

在实际工作中你有没有遇到命令什么时候完成？

通常会点击 RETURN 几次以确保它没有崩溃，或者SSH连接没有冻结吗？你有没有想过能够毫不费力地暂停一些处理，返回 Python 提示符手动修复一些项目，然后无缝地恢复它会很棒？这个工具做到了
