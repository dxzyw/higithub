### 文件和目录操作

| 命令 | 描述 | 重要参数 |
| --- | --- | --- |
| `ls` | 列出目录内容 | `-l`（详细列表）、`-a`（显示隐藏文件） |
| `cd` | 切换目录 | 无 |
| `pwd` | 显示当前工作目录 | 无 |
| `touch` | 创建空文件 | 无 |
| `mkdir` | 创建目录 | `-p`（递归创建） |
| `cp` | 复制文件或目录 | `-r`（递归复制）、`-i`（交互式复制） |
| `mv` | 移动/重命名文件或目录 | 无 |

| `find` | 搜索文件 | `-name`（按文件名搜索）、`-type`（按类型搜索） |
