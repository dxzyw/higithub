### **Chatlog 简介**  
Chatlog 是一款强大的聊天记录管理工具，专为微信用户设计，能够帮助你轻松使用自己的聊天数据。它支持 Windows 和 macOS 系统，可从本地数据库文件获取聊天记录，并提供多种查询、管理方式，确保你的信息井然有序，使用更高效。

### **功能特点**  
1. **数据管理便捷**——Chatlog 可从本地数据库提取聊天记录，支持解密图片、语音等多媒体消息，让你轻松查找和回顾重要对话。  
2. **强大的搜索和查询能力**——提供 HTTP API 服务，可查询聊天记录、联系人、群聊、最近会话等信息，使查找更加精准。  
3. **支持 MCP SSE 协议**——能够与 MCP 兼容的 AI 助手无缝集成，实现更智能的交互体验。  
4. **自动解密数据**——简化使用流程，让你无需复杂操作即可访问和管理聊天记录。  
5. **多账号管理**——支持在不同微信账号之间切换，满足多账号用户的需求。  
6. **多种访问方式**——不仅可以通过 Terminal UI 界面操作，还支持命令行模式，方便不同习惯的用户使用。  

### **如何快速开始？**  
想要使用 Chatlog？只需简单几步即可轻松上手：
1. **安装 Chatlog**  
   - 你可以访问[官方发布页面](https://github.com/sjzar/chatlog)下载适合自己系统的预编译版本，或者使用 Go 直接安装：
     ```
     go install github.com/sjzar/chatlog@latest
     ```
2. **运行程序**  
   - 执行 `chatlog` 启动 Terminal UI 界面。  
3. **解密数据**  
   - 选择“解密数据”菜单项，确保你的聊天记录可以正常访问。  
4. **开启 HTTP 服务**  
   - 选择“开启 HTTP 服务”，通过 API 访问聊天记录。  
5. **访问聊天数据**  
   - 通过 HTTP API 或 MCP 集成，可以随时查看聊天内容，快速检索关键信息。
