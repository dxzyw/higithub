🛠️ 五分钟极速上手  
1️⃣ 安装神器（支持三大系统）  
```bash  
Mac用户专属  
brew install bunster  

通用安装方式  
curl -fsSL https://bunster.io/install | bash
```

2️⃣ 编译初体验  
把老旧脚本改造成超级程序：  
```bash  
bunster build deploy.sh -o deploy_tool
```

3️⃣ 跨平台炫技时刻  
直接将生成的`deploy_tool`扔给Windows同事——零配置直接运行！
